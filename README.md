TITLE: TrendWear: Clothing shop app
Description: The TrendWear Clothing Shop App is a simple mobile shopping application
that helps users easily explore clothing items such as T-shirts, hoodies, and pants, and
place orders online. Customers can view available products, check product details, add
items to their cart, and proceed to checkout. On the admin side, products can be added,
updated, or removed, making it easy to keep the shop organized and updated. The system
is developed using React Native and Firebase Firestore, which serves as the database for
storing product and order information in real time. This project focuses on basic CRUD
(Create, Read, Update, Delete) operations, making it suitable for beginners who want to
understand how a simple e-commerce system works.
The system is divided into two main users: the admin and the customers. The admin
manages the products by performing CRUD operations such as adding new items, editing
product details, viewing the product list, and deleting unavailable products. All product
information is stored in the Firebase Products collection. Meanwhile, customers can
browse the available clothing items, view detailed product information, add selected items
to their cart, and proceed to checkout. Once an order is placed, the order details are saved
in the Firebase Orders collection.
Detailed System Flow
1. Admin logs in to the app.
2. Admin adds products (name, price, description, image).
3. Products are saved in the Firebase Products collection.
4. Users open the app and browse available clothing items.
5. Users can view product details.
6. Users add products to cart.
7. Users proceed to checkout.
8. Order information is saved in the Firebase Orders collection.
9. Admin can update or delete products anytime.
